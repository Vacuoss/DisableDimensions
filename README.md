This plugin can disable or enable any dimension, from vanilla,mods and data-packs
Can work on Papper,Bukkit,Spigot,Follia,Mohist,Purpur on version: 1.19.1-1.20.6
How to use:
/dimensions <nether/end/custom> <on/off>
/dimensions reload
Example:
/dimensions twilightforest:twilight_forest off
/dimensions reload
