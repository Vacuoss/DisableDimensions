Plugin to disable vanilla, modded, or datapack dimensions with simple commands.
Block portals, teleportation, and customize all messages from the config
Features:

Disable the Nether

Disable The End

Disable custom / modded dimensions

Prevent teleportation into restricted worlds

Fully configurable messages

Hot reload: /dimensions reload

List all loaded dimensions

Tab-complete support
